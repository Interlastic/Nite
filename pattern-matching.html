Quick Start - Most Common Patterns

(?i)hello Match "hello" ignoring uppercase/lowercase
Example: (?i)hello ‚Üí matches "hello", "HELLO", "HeLLo"
\p{Extended_Pictographic} Match any emoji
Example: \p{Extended_Pictographic} ‚Üí matches üòÄ, üéâ, üî•, üíÄ, etc.
\bword\b Match whole word only (not inside other words)
Example: \bcat\b ‚Üí matches "cat" but NOT "category" or "scat"
(a|b|c) Match any one of multiple options
Example: (cat|dog|bird) ‚Üí matches "cat" or "dog" or "bird"

Case & Mode Flags

(?i) Case-insensitive matching (ignore CAPS)
Example: (?i)lol ‚Üí matches "lol", "LOL", "LoL", "lOl"
(?m) Multiline mode (^ and $ work per line)
Example: (?m)^hello ‚Üí matches "hello" at start of ANY line in message
(?s) Dotall mode (. matches newlines too)
Example: (?s)start.*end ‚Üí matches text from "start" to "end" even across lines
(?x) Verbose mode (ignore spaces, allow comments)
Example: (?x) hello # this is a comment ‚Üí matches "hello" (spaces/comments ignored)
(?-i) (?-m) (?-s) Turn OFF a flag
Example: (?i)hello(?-i)WORLD ‚Üí "hello" case-insensitive, but "WORLD" must be exact

Quantifiers (How Many Times)

* Zero or more times
Example: lo*l ‚Üí matches "ll", "lol", "loool", "looooool"
+ One or more times
Example: lo+l ‚Üí matches "lol", "loool" but NOT "ll"
? Zero or one time (optional)
Example: colou?r ‚Üí matches both "color" and "colour"
{n} Exactly n times
Example: ha{3} ‚Üí matches "haaa" only
{n,} n or more times
Example: ha{2,} ‚Üí matches "haa", "haaa", "haaaa", etc.
{n,m} Between n and m times
Example: ha{2,4} ‚Üí matches "haa", "haaa", "haaaa" (2-4 a's)
*? +? ?? Lazy (match as few as possible)
Example: <.+?> ‚Üí in "<b>hi</b>" matches "<b>" then "</b>" separately

Anchors (Position in Text)

^ Start of message
Example: ^hello ‚Üí matches only if message STARTS with "hello"
$ End of message
Example: bye$ ‚Üí matches only if message ENDS with "bye"
\b Word boundary (edge of a word)
Example: \bhello\b ‚Üí matches "hello" but NOT "helloworld" or "sayhello"
\B NOT a word boundary
Example: \Bcat ‚Üí matches "cat" in "scat" but NOT in "cat food"
\A Absolute start (ignores multiline)
Example: \Ahello ‚Üí matches "hello" only at very beginning
\Z Absolute end (ignores multiline)
Example: bye\Z ‚Üí matches "bye" only at very end

Groups (Capture & Organize)

(pattern) Capturing group - saves what it matches
Example: (hello) world ‚Üí matches "hello world", captures "hello"
(?:pattern) Non-capturing group (just for grouping)
Example: (?:ha)+ ‚Üí matches "ha", "haha", "hahaha" (groups "ha" but doesn't capture)
(?P<name>pattern) Named group - give it a name
Example: (?P<word>\w+) ‚Üí captures any word and names it "word"
\1 \2 \3 Backreference - match same thing again
Example: (\w+) \1 ‚Üí matches repeated words like "the the" or "hello hello"
(?>pattern) Atomic group (no backtracking)
Example: (?>a+)b ‚Üí matches "aaab" but faster (won't retry)

Lookahead & Lookbehind (Check Without Matching)

(?=pattern) Positive lookahead - must be followed by
Example: hello(?= world) ‚Üí matches "hello" only if followed by " world"
(?!pattern) Negative lookahead - must NOT be followed by
Example: hello(?! world) ‚Üí matches "hello" only if NOT followed by " world"
(?<=pattern) Positive lookbehind - must come after
Example: (?<=@)\w+ ‚Üí matches username in "@username" (without the @)
(?<!pattern) Negative lookbehind - must NOT come after
Example: (?<!un)happy ‚Üí matches "happy" but NOT "unhappy"

Alternation (OR)

a|b Match a OR b
Example: yes|no ‚Üí matches "yes" or "no"
(a|b|c) Match any of multiple options
Example: (?i)(lol|lmao|rofl) ‚Üí matches any laugh (case-insensitive)

Character Classes (Match Specific Characters)

[abc] Match a, b, or c
Example: gr[ae]y ‚Üí matches "gray" or "grey"
[a-z] Match any letter from a to z
Example: [a-z]+ ‚Üí matches any lowercase word like "hello"
[0-9] Match any digit
Example: [0-9]{3} ‚Üí matches any 3 digits like "123", "456"
[^abc] Match anything EXCEPT a, b, c
Example: [^0-9]+ ‚Üí matches anything that's NOT a number
[a-zA-Z0-9] Match letters and numbers
Example: [a-zA-Z0-9]+ ‚Üí matches "Hello123", "test", "ABC"

Shorthand Classes (Quick Shortcuts)

\d Any digit (same as [0-9])
Example: \d{4} ‚Üí matches "2024", "1234"
\D NOT a digit
Example: \D+ ‚Üí matches "hello", "!!!", " " (no numbers)
\w Word character (letters, numbers, underscore)
Example: \w+ ‚Üí matches "hello_world", "user123"
\W NOT a word character
Example: \W+ ‚Üí matches "!!!", "@#$", " - "
\s Whitespace (space, tab, newline)
Example: hello\s+world ‚Üí matches "hello world", "hello world"
\S NOT whitespace
Example: \S+ ‚Üí matches any word without spaces
. Any character (except newline)
Example: h.t ‚Üí matches "hat", "hit", "hot", "h@t"

Conditional Patterns (If/Then)

(?(1)yes|no) If group 1 matched, use "yes", else "no"
Example: (<)?hello(?(1)>) ‚Üí matches "hello" or "<hello>" (balanced)

Fuzzy Matching (Allow Typos!)

(?:pattern){e<=n} Allow up to n total errors
Example: (?:hello){e<=2} ‚Üí matches "hello", "helo", "helllo", "hallo" (2 errors max)
(?:pattern){i<=n} Allow up to n insertions
Example: (?:hello){i<=1} ‚Üí matches "hello", "helllo" (1 extra letter)
(?:pattern){d<=n} Allow up to n deletions
Example: (?:hello){d<=1} ‚Üí matches "hello", "helo" (1 missing letter)
(?:pattern){s<=n} Allow up to n substitutions
Example: (?:hello){s<=1} ‚Üí matches "hello", "hallo" (1 wrong letter)
(?b) Best match mode
Example: (?b)(?:hello){e<=3} ‚Üí finds the closest match to "hello" possible

Recursion (Advanced)

(?R) Repeat the whole pattern
Example: \((?:[^()]+|(?R))*\) ‚Üí matches nested parentheses "(a(b(c)d)e)"
(?1) Repeat group 1
Example: (\w)(?1)? ‚Üí matches "a", "ab", "abc"

Special Escapes

\n Newline (line break)
Example: line1\nline2 ‚Üí matches text with newline between
\t Tab character
Example: a\tb ‚Üí matches "a[TAB]b"
\\ Literal backslash
Example: C:\\Users ‚Üí matches "C:\Users"
\. \* \+ \? \[ \] Escape special characters
Example: what\? ‚Üí matches "what?" (literal question mark)
\( \) \{ \} \| \^ \$ More escaped characters
Example: \(hello\) ‚Üí matches "(hello)" with actual parentheses
